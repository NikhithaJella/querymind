## Project-specific context

QueryMind was built as a self-service analytics prototype for CSV-based business analysis. The main workflow includes CSV upload, DuckDB table creation, data profiling, Text-to-SQL generation, SQL validation, query execution, chart generation, anomaly detection, and report export.

I can explain how the system moves from uploaded CSV data to schema detection, SQL generation, query execution, visual insights, and user-facing analytics output.
